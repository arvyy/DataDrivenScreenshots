2 (dds-interp) module reference

This module provides interpolation between different values. Module is included by default.

2.1 interp-bridge



(interp-bridge
        #:id id
        #:enter enter
        #:update update
        #:exit exit)

  • #:id id - function that returns id for item, by which it’s determined the items that entered, stayed, or exited. If id is #f, index is used. (Default: #f)

  • #:enter enter - function for entering items. (Default: identity)

  • #:update update - function for update items. (Default: identity)

  • #:exit exit - function for exit items. (Default: identity)


Returns function, that takes two lists as argument, and returns two lists as bridging result

2.2 interp



(interp a b f [ extra-dispatch ] )

  • a - first value to interpolate from

  • b - second value to interpolate towards

  • f - interpolation factor

  • optional extra-dispatch - additional cases for interpolation. Must be a pair of two functions. First function takes a and b, and returns #t if it’s suitable for interpolation. Second function takes does the interpolation; its arguments are same as to this base one. (a, b, f, extra-dispatch).( Default: #f)


Returns interpolated value

2.3 ease



(ease f)

  • f - value from 0 to 1 to be eased


Returns eased value

2.4 start-end

Maps range [0, 1] to [start, end]. Start and end must be within range [0,1]

(start-end start end f)

  • start - lower bound

  • end - upper bound

  • f - value [0 - 1]


Returns interpolated value
