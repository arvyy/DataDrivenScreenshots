4 (dds-bezier) module reference.

This module provides bezier interpolation, as well as bezier line construction. You have to include module explicitly to use.

4.1 bezier

Create a line item definition using bezier control points. The degree depends on the amount of points provided.

(bezier
        #:id id
        #:points points
        #:segments segments
        #:stroke stroke
        #:stroke-width stroke-width)

  • #:id id - id; used in override. (Default: #f)

  • #:points points - list of points. Each point is a pair (cons) of numbers. Only first and last points are generally on the curve, middle ones are control points. (Default: empty list)

  • #:segments segments - amount of lines used to render the curve. (Default: 20)

  • #:stroke stroke - color to stroke shape, or #f to disable. (Default: (color 10 10 10))

  • #:stroke-width stroke-width - thickness of the stroke. (Default: 1)


Returns line item definition.

4.2 bezier-interp/quadratic

Interpolate value using quadratic bezier function

(bezier-interp/quadratic p0 c p1 t)

  • p0 - starting value

  • c - control value

  • p1 - ending value

  • t - coeficient along the curve, must be in range [0, 1]


Returns interpolated value

4.3 bezier-interp/cubic

Interpolate value using cubic bezier function

(bezier-interp/cubic p0 c0 c1 p1 t)

  • p0 - starting value

  • c0 - first control value

  • c1 - second control value

  • p1 - ending value

  • t - coeficient along the curve, must be in range [0, 1]


Returns interpolated value

4.4 bezier-interp/n

Interpolate value using any degree bezier function (degree depends on amount of points provided).

(bezier-interp/n points t)

  • points - list of points

  • t - coeficient along the curve, must be in range [0, 1]


Returns interpolated value
